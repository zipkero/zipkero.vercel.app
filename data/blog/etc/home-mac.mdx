## 설정

- ssh-key

```bash
ssh-keygen
```

- ssh 로 github push 가 되지 않는 경우 .ssh 폴더에 config 파일 생성 후 아래와 같이 기입

```
Host github.com
    Hostname ssh.github.com
    Port 443
```

- 터미널 단축키 .zshrc

```
alias k="kubectl"
alias ll="ls -al"
alias tf="terraform"
```
