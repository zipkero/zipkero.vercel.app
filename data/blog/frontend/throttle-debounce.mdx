### throttle

이벤트 발생 후 다시 이벤트가 발생하더라도 지정된 시간이 지나지 않았으면 실행이 되지 않도록 하는 방식

```javascript
const throttle = (func, delay) => {
  let throttled = false
  return (...args) => {
    if (!throttled) {
      throttled = true
      setTimeout(() => {
        func(...args)
        throttled = false
      }, delay)
    }
  }
}
```

### debounce

여러번의 이벤트가 발생하는 경우 마지막 이벤트만 처리가 되는 방식

```javascript
const debounce = (func, delay) => {
  let timeoutId = null
  return (...args) => {
    clearTimeout(timeoutId)
    timeoutId = setTimeout(func.bind(this, ...args), delay)
  }
}
```
