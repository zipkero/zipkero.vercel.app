1. 솔루션 생성

```bash
dotnet new sln -n {solutionName}
```

2. 솔루션 프로젝트 리스트

```bash
dotnet sln {solutionName}.sln list
```

3. 프로젝트 생성

```bash
# dotnet new --list 로 확인 필요
dotnet new [console,classlib,web,mvc,react] -n {projectName}
# sdk 버전 지정
dotnet new console -n {projectName} -f net6.0
```

4. 솔루션에 프로젝트 추가, 삭제

```bash
dotnet sln add todo-app/todo-app.csproj
dotnet sln remove todo-app/todo-app.csproj
```

5. 프로젝트 참조

```bash
# 지정된 프로젝트에 참조 추가
dotnet add app/app.csproj reference lib/lib.csproj

# 현재 프로젝트에 지정된 프로젝트들을 참조
dotnet add reference lib1/lib1.csproj lib2/lib2.csproj
```

6. 빌드

```bash
dotnet build
# 빌드 프로젝트 지정
dotnet build path/project.csproj
# 빌드 구성 지정
dotnet build --configuration [Debug|Release]
# 빌드 런타임 지정
dotnet build --runtime [win10-x64|linux-x64|osx-x64|...]
# 출력 디렉토리 지정
dotnet build --output /path/output-directory
```

7. 실행

```bash
dotnet run
dotnet run --project ./projects/proj1/proj1.csproj
```
